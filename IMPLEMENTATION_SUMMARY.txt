================================================================================
  HOW TO ADD MORE COMPONENTS
================================================================================

Using MCP (Recommended):
  1. Ask Claude to list available components
  2. Get component source via MCP
  3. Save to src/components/ui/[name].tsx
  4. Import and use: import { ComponentName } from "@/components/ui/[name]"

Using CLI:
  1. npx shadcn-ui@latest add [component-name]
  2. Component auto-added to src/components/ui/

Example:
  npx shadcn-ui@latest add card
  npx shadcn-ui@latest add dialog
  npx shadcn-ui@latest add form
